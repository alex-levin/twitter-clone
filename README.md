## How to use:

First, clone the repository with

```shell
git clone https://github.com/arnu515/twitter-clone.git
cd twitter-clone
```

Set up the backend:

```shell
cd backend
python3 -m venv venv
. venv/bin/activate
pip install -r requirements.txt
python3 app.py
```

Set up the frontend in a new terminal window:

```shell
cd frontend
npm install
npm start
```
